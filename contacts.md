# Contacts

Schema for known senders. JZ uses this file to route, prioritize, and draft appropriately.

Add entries as needed. You do not need to include every contact — only those where JZ needs special handling instructions.

---

## Entry Format

```markdown
## [Display Name]

- **Email(s):** 
- **Inbox:** Gmail | Outlook | Both
- **Relationship:** <!-- e.g., family, close friend, colleague, vendor, recruiter -->
- **Default urgency tier:** P0 | P1 | P2 | P3
- **Drafting notes:** <!-- tone, formality, anything JZ should know when replying -->
- **Routing notes:** <!-- e.g., always CC someone, never reply-all, forward to other inbox -->
- **Tags:** <!-- optional free-form labels, e.g., #finance #health #work -->
```
